
------------------------- ENTORNO VIRTRUAL --------------------------------------------

paso 1: crear un entorno virtual, para instalar las dependencias(requirements.txt)

paso 2: entrar en modo de entorno virtual, comando: ./envs/Scripts/activate

paso 3: instalar las dependencias con el comando: pip install -r requirements.txt

paso 4: verificar que esten instalas todas las dependencias con el comando: pip list

------------------------- CUENTA TWILIO --------------------------------------------

paso 1: crear una cuenta en Twilio.com

paso 2: copiar y pegar las credenciales de Twilio(Sid & Token) para usarlas en el
	codigo de Python (main.py).

paso 3: escanear el QR de Twilio para hacer conectividad.

paso 4: definir el numero de Telefono de Twilio en el codigo, asi como 
	tambien el numero de WhatsApp que queremos enviar el msg en el
	codigo de Python(main.py).
